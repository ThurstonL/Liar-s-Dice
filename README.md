# Liar's Dice

A multiplayer Liar's Dice web application.

## Prerequisites

- Node.js (version 18 or higher recommended)
- npm package manager

## Installation

Install dependencies for the workspace (this covers client, server, and shared directories):

```bash
npm install
```

## Running Locally

To start both the frontend (client) and backend (server) in development mode simultaneously:

```bash
npm run dev
```

- The client will typically be running on [http://localhost:5173](http://localhost:5173) (or similar, depending on your Vite/React setup).
- The server will be running on its assigned port (often 3000 or similar).

## Building for Production

To build both the client and server for production:

```bash
npm run build
```

## Running Production Build

After building, you can start the production server:

```bash
npm start
```
